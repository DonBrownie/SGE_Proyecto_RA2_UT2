# 02 — Requisitos previos


###  Sistema operativo

- Windows 10 u 11 (64 bits)
- Permisos de administrador

### Requisitos mínimos

- Espacio libre: 5 GB
- Memoria RAM: 4 GB mínimo (8 GB recomendado)
- Conexión a Internet

### Herramientas necesarias

- Navegador para descargar recursos
- Desinstalar versiones antiguas de Odoo si existen
- Instalar **PostgreSQL** (si el instalador de Odoo no la incluye)
